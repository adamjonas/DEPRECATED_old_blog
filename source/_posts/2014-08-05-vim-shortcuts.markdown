### File Search
  - cntl + p => file search

### Nerd Tree
  - `go` => preview pane

### Run the specs
  - `\r` runs the closest spec
  - `\R` runs the whole spec suite

### cmd + shift + d
  - `control + n`

### vimium for life
  - `\\f` + character you are looking for
  - `\\w` + word
  - `\\e` goes to end of words
  - you get it

### Go to line
  - `line number + G`

### Tab in or out
  - `>>` & `<<`
