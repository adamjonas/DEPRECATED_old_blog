***Rake***
- Rake lets you define a dependency tree of tasks to be executed.
- Rake tasks are loaded from the file Rakefile
- Rails rake tasks live under railties/lib/tasks
- You can put your own tasks under lib/tasks


Useful Rake Tasks
```
    db:migrate
    db:sessions:create
    doc:app
    doc:rails
    log:clear
    rails:freeze:gems
    rails:freeze:edge
    test (default task)
    stats
```
